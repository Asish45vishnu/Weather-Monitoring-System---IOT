# Weather Monitoring System - IOT
Hardware Used :
- NODEMCU (ESP8266 WiFi Module)
- DHT 11(Temperature and Humidity Sensor)
- BMP 180 (Pressure Sensor)
  
Software:
- Thingspeak by matlab
